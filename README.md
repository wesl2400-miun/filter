# Frontend-baserad webbutveckling med JavaScript
## Laboration 2
Laborationen går ut på att skapa en webbplats som visar en tabell över 
Mittuniversitetets kurser i datateknik. Tabellen ska kunna sorteras efter
olika kategorier. Kursinformationen baseras på JSON-data som
i uppgiften ska hämtas asynkront via metoden fetch().
